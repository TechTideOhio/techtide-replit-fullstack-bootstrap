# techtide-replit-fullstack-bootstrap

Single-skill Lovable import repository from [TechTide Harness Kit](https://github.com/TechTideOhio/techtide-harness-kit).

Use this URL in Lovable Settings, Skills, Import, GitHub:

```
https://github.com/TechTideOhio/techtide-replit-fullstack-bootstrap
```

This repository intentionally contains one SKILL.md package so Lovable can import it without reading the full TechTide Harness Kit catalog.
